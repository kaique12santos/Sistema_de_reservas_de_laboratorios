# Sistema de Gestão e Reserva de Laboratórios Acadêmicos

Projeto desenvolvido no 4º semestre do curso de Desenvolvimento de Software Multiplataforma (DSM) com o objetivo de modernizar o processo de reserva de laboratórios da FATEC Zona Leste.

O sistema substitui uma solução legada em PHP por uma arquitetura mais organizada, escalável e alinhada com boas práticas modernas de desenvolvimento web.

---

## 🎯 Objetivo

Criar uma plataforma web para:

- Gerenciamento de laboratórios
- Solicitação de reservas
- Fluxo de aprovação hierárquico
- Controle de acesso por perfil
- Visualização organizada de agendas e reservas

---

## 🏗 Arquitetura

O projeto será estruturado em três camadas principais:

- **Frontend:** React + Material UI
- **Backend:** Node.js + Express
- **Banco de Dados:** PostgreSQL
- **Containerização:** Docker

A organização será modular, com separação clara entre regras de negócio, interface e persistência de dados.

---

## 👥 Perfis de Acesso (Planejado)

- Administrador
- Coordenador
- Professor
- Usuário solicitante

Cada perfil terá permissões específicas dentro do fluxo de reservas.
